# sentiment-analysis
Comprehensive Sentiment Analysis project using three Kaggle datasets (Twitter Airline, IMDb Movies, and Amazon Reviews). Trains Logistic Regression, Naive Bayes, and SVM models, combines them into an ensemble, compares accuracies, and visualizes performance.
# Triple Dataset Sentiment Analysis (Twitter + IMDb + Amazon)

This project performs sentiment analysis using three datasets from Kaggle:
- Twitter Airline Sentiment
- IMDb Movie Reviews
- Amazon Product Reviews

## Features
- Text cleaning + TF-IDF vectorization
- Three models: Logistic Regression, Naive Bayes, SVM
- Ensemble model for each dataset
- Accuracy comparison chart across all datasets
- Models saved to Google Drive for reuse

## How to run
1. Upload kaggle.json in Colab
2. Run all cells in order
3. View accuracy & confusion matrices
4. Save notebook to GitHub
